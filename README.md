# 📚 DSA — Data Structures & Algorithms Practice

A personal repository by **Doyel Mishra** for practising Data Structures and Algorithms. This repo is used to keep track of solved problems, implementations, and notes across a wide range of DSA topics.

---

## 🗂️ Topics Covered

| #  | Topic                        | Description                                              |
|----|------------------------------|----------------------------------------------------------|
| 1  | Arrays                       | Traversal, searching, sorting, sliding window, two pointers |
| 2  | Strings                      | Pattern matching, manipulation, anagrams, palindromes    |
| 3  | Linked Lists                 | Singly, doubly, circular; reversal, cycle detection      |
| 4  | Stacks & Queues              | Monotonic stack, deque, priority queue problems          |
| 5  | Hashing                      | HashMaps, HashSets, frequency counting                   |
| 6  | Recursion & Backtracking     | Subsets, permutations, N-Queens, sudoku solver           |
| 7  | Sorting & Searching          | Merge sort, quick sort, binary search variants           |
| 8  | Trees                        | Binary trees, BSTs, AVL trees, tree traversals           |
| 9  | Heaps                        | Min/Max heap, heap sort, K-th largest/smallest           |
| 10 | Graphs                       | BFS, DFS, Dijkstra, Topological sort, Union-Find         |
| 11 | Dynamic Programming          | Memoization, tabulation, knapsack, LCS, LIS              |
| 12 | Greedy Algorithms            | Activity selection, interval scheduling, Huffman coding  |
| 13 | Tries                        | Insert, search, prefix queries, word dictionaries        |
| 14 | Segment Trees & Fenwick Trees| Range queries and point/range updates                    |
| 15 | Bit Manipulation             | XOR tricks, bit masking, counting set bits               |
| 16 | Mathematics                  | GCD/LCM, prime sieve, modular arithmetic, combinatorics  |
